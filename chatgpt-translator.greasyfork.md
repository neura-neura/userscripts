# ChatGPT Translator

A compact translator control for ChatGPT that feels native to the composer.

## Features

- Toggle translation mode on or off from the composer.
- Choose source and target languages with searchable custom dropdowns.
- Use automatic source language detection.
- Pin frequent languages for quick access.
- Type a custom language when it is faster than searching the list.
- Saves enabled state, selected languages, custom languages, and pinned languages in `localStorage`.
- Automatically injects a translation-only prompt when you send a message.
- Supports light and dark themes.

## Usage

Open ChatGPT, enable **Translator**, choose the source and target languages, type only the text you want translated, and send the message. ChatGPT will receive the translation prompt automatically and should reply only with the translated text.
