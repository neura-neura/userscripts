# ChatGPT Extra Tools

Adds a floating **Extra tools** button to ChatGPT with reusable prompt helpers that can be toggled independently from ChatGPT's changing interface.

## Features

- Adds a persistent floating Extra tools button in the bottom-right corner.
- Includes a subtitle translator helper for SRT, ASS/SSA, VTT, TTML, and DFXP workflows.
- Includes a general translator helper with source and target language controls.
- Shows a floating active-tools label when a helper is enabled.
- Includes a persistent switch to disable active tools automatically on page reload.
- Saves tool settings locally in the browser.
- Injects the selected professional prompt automatically when you send a message.

## Usage

Open ChatGPT, click the floating **Extra tools** button in the bottom-right corner, select the helper you want, adjust its settings, and send your message normally. The script prepares the prompt locally before ChatGPT receives it.

If you want every page reload to start with no active helper, turn on **Disable active tools on reload** from the Extra tools menu.
