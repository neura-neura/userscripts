# ChatGPT Extra Tools

Adds an **Extra tools** menu to ChatGPT with reusable prompt helpers that can be toggled directly from the composer.

## Features

- Adds an Extra tools submenu to ChatGPT's attachment/tools menu.
- Includes a subtitle translator helper for SRT, ASS/SSA, VTT, TTML, and DFXP workflows.
- Includes a general translator helper with source and target language controls.
- Shows the active tool as a composer pill so it is easy to see when a helper is enabled.
- Saves tool settings locally in the browser.
- Injects the selected professional prompt automatically when you send a message.

## Usage

Open ChatGPT, choose **Extra tools** from the composer tools menu, select the helper you want, adjust its settings, and send your message normally. The script prepares the prompt locally before ChatGPT receives it.
